--------------------------

STEP 1 — Upload the sales page to the repo ROOT

  5. Open GITHUB_ROOT_index.html in Notepad or VS Code

  7. Click back in GitHub editor → Ctrl+V to paste

STEP 2 — Upload the app to the app/ folder

STEP 3 — Verify GitHub Pages is enabled
